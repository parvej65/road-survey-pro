🚀Quick Start

1. Login with your username & password (or tap a user tile).
2. App opens directly on the Survey panel.
3. Go to Road Info first and fill in the Road Name (required before tracking).
4. Return to Survey, press ▶ AUTOMATIC to start logging your route as you walk/drive.
5. Use 📍 MANUAL POINT any time to record a feature (culvert, bridge, damage, etc.) with GPS averaging.
6. Press ⏹ STOP when the survey is complete.
7. Go to Export to download KML / CSV / Report, and back up your session.

📡1. Survey Panel

GPS Signal Quality — shows live accuracy (in metres), satellites, altitude, speed and heading. Green = good signal, yellow = fair, red = poor. Wait for green/yellow before marking points.

Map — switch between Satellite, Street and Topo views using the dropdown. Tap 🎯 Center to jump the map to your current location.

Interval (m) — set the distance (1–500 metres) between automatically captured points before starting tracking, e.g. enter 5 for a point every 5 m.

▶ AUTOMATIC — starts continuous GPS tracking. The app automatically logs route points at the interval you set above as you move, and the map auto-follows your live location. Road Name must be set first (in Road Info).

⏹ STOP — stops automatic tracking and saves the session.

📍 MANUAL POINT — captures a single point at your current location using GPS averaging (set the number of samples below — more samples = higher accuracy but takes longer). Use this for chainage marks, structures, damage, landmarks etc.

Mark Point Settings — choose the Point Type, add a Remark, and set Averaging Samples before tapping Manual Point.

Photo Capture — take a GPS-stamped photo of the current location; it's saved with chainage and accuracy info.

🗂️2. Road Info Panel

Fill in the road's identity (name, code, type, from/to, district, state, block/GP), structural dimensions (widths, lanes, embankment), surface & condition, drainage & culverts/bridges, terrain & soil, and project/admin details (scheme, agreement no., cost, priority, remarks).

This information is saved automatically and appears in the final Road Attribute Report and KML description. Road Name is required before you can start Automatic tracking.

✏️3. Correct Panel

Use this to fix GPS drift after a survey.

🔄 Load Draggable Points — loads every recorded point as a draggable marker on the map.
Drag any marker to its correct position — changes are saved automatically and the route line updates instantly.
🗑️ Delete Last — removes the most recently recorded point.
✖ Clear Markers — hides the draggable markers (does not delete data).
🎯 Fit All — zooms the map to fit the whole survey.

You can also switch between Satellite, Street and Topo views here, same as the Survey map.

Tip: Correction is easiest on a PC/laptop with a mouse for precision dragging.

📋4. Log Panel

Shows every recorded point with coordinates, accuracy and type. Use All / Marked / Route filters to narrow the list — "Marked" shows only manually-recorded points, "Route" shows only automatic tracking points.

Chainage Segments — lists each marked point with its chainage and distance from the previous one, useful for reviewing structure spacing along the road.

⬇️5. Export Panel

☁️ Backup & Sync — "Upload Backup" downloads a full session JSON file (all points, photos & road info) that you can save to Drive/storage; "Fetch / Load" lets you pick a previously saved JSON file to restore a session.

🗺️ KML Export — "Centerline KML" exports just the road alignment (curve-aware, smoothed for Google Earth); "Full Survey KML" also includes all marked points with icons and details.

📑 Data Export — download all points or only marked points as CSV, or generate a full Road Attribute Report (HTML) combining Road Info with the marked-points table.

🗑️ Reset / New Survey — clears all data on this device and starts fresh. This cannot be undone — export/backup first!

💡Tips

• Keep your phone's GPS / Location set to "High Accuracy" mode.
• Survey outdoors with a clear sky view for best accuracy.
• Use higher averaging samples (10–20) for important fixed points like culverts and bridges.
• Back up your session regularly using "Upload Backup" in the Export panel.
• If the map shows blank tiles when zoomed in very far, zoom out one or two levels — satellite imagery isn't available at the deepest zoom in every area.
